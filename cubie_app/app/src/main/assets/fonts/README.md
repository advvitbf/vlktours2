# Optional Nothing-style fonts

The app will automatically use licensed font files placed in this folder:

```text
NType82-Regular.ttf
NType82-Medium.ttf
NType82Mono-Regular.ttf
LLLetteraMono-Regular.ttf
NDot57.ttf
NDot55.ttf
```

These are custom/proprietary Nothing brand typefaces, so they are not bundled here. If no licensed font files are present, the app falls back to Android system fonts while keeping the Nothing-inspired layout, spacing, contrast, and dot-label treatment.
